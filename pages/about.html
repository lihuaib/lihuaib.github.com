我的故事

就像很多家庭的孩子一样， 我于88年出生在一个小镇，7岁上小学，13岁上初中，17岁上高中， 20岁上大学，24岁(2011年)毕业于一家普通的大学， 然后就去上班了.

迄今为止， 我服务过的公司有宁波的CI&T(宁波乐科科)一家日巴合资的企业，主要做CMS， 用过Sharepoint, 也用过drupal, 接触过敏捷开发， jira等等项目管理工具， 我想， 这对我的整体项目理念的形成和实践起了很大的帮助.

忽然有一天，我觉的似乎在CI&T 我技术无法得到提高，我想，该出去做做更难更有挑战的项目了。于是我就去了杭州的感遇科技，去做游戏.

我知道做游戏的强度很大， 但是事实比我想的更加高难度，每天早上10点上班，晚上11点下班似乎已经是家常便饭。在其中，我一边做服务器photon开发，一边学习Unity做前台开发，虽然累， 但也很充实， 就这样，我们不到3个月就开发出了康熙德州扑克，然后兴致勃勃的在腾讯上首发。

所谓希望是美好的， 结果是残酷的，因为我们的游戏风格跟博雅太相似， 经过几个月的营销，终于业绩惨败，背后财团开始撤资， 然后公司就宣布解散了。

于是我开始了自己自己对过去的思考， 也开始看些别的书， 去爬爬山，户外运动, 空闲时间也去拜访原来的老师，然后就接接项目做做。

也是在这段时间， 我开始认真读Sharpdevelop的源代码，开始总结以前的写作手法和技巧，也开始了在项目中有选择的选择需要的库，并且都会对所有的代码进行简单的查看， 不是以前的拿来主义了。

时间到了2013年的7月，绍兴易通软件突然打了个电话给我，我去面试了下，然后加个谈好就进去了，做的是php + java + Linux + android 的开发和管理， 跟以前的开发方式也大致相同 。。。。。。
